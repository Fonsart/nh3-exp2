# nh3-exp1

## Project setup

```bash
npm install
```
### Run project for dev

```bash
npm start
```

### Run project through HTTPS

```bash
npm run https
```

### Compiles and minifies for production

```bash
npm run build
```

### Serve local build
requires `npm install -s serve`
### Run your tests
```bash
serve -s build
or
npx serve -s build
```

```bash
npm run test
```
## Images setup
Images used to generate the mosaic are located in: 
```bash
public/images
```
JSON file served to ReactImageMosaic is located in:
```bash
src/assets/images.full.json
```
